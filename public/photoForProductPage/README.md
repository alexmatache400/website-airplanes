### Image Requirements

- **Format:** JPG, PNG, or WebP
- **Recommended Size:** 800x600px or higher
- **Aspect Ratio:** 4:3 or 16:9
- **File Size:** Keep under 500KB for optimal loading

### Fallback Behavior

If images are missing or fail to load, the site will automatically display a fallback SVG joystick/throttle icon.

### Testing

After adding images:
1. Restart the dev server (Ctrl+C, then `npm start`)
2. Navigate to http://localhost:3000/products
3. Images should load automatically from this directory
